Of course! Substitution is a core skill for the CEMAC exams. Here is the logic:

  • Isolate one variable in one of the equations (e.g., x = ...).
  • Substitute that expression into the other equation.
  • Solve for the remaining variable.
  • Plug back to find the first variable.
